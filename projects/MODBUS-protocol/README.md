# Реализация MODBUS-управления 8 лампочками через UART

## Задание

Разработать программу для микроконтроллера (МК), которая позволяет через COM-порт (UART) по протоколу **MODBUS RTU** управлять восемью дискретными выходами (лампочками). В роли ведущего устройства (Master) выступает терминал (ПК), в роли ведомого (Slave) – МК.

### Требования к функционалу

- МК подключён к 8 лампочкам.
- Связь с терминалом осуществляется через UART с параметрами: `Скорость: 9600 бод`, `8 бит данных`, `без чётности`, `1 стоп-бит`.
- Адрес ведомого устройства (Slave ID) зафиксировать, например, `0x01`.
- Обработка ошибок (только два типа): неверный адрес, неверный регистр.
- При ошибке МК не меняет состояние лампочек.

### Требования к реализации

- Не использовать сторонние MODBUS-библиотеки (реализовать протокол самостоятельно).
- Обеспечить устойчивость к повреждённым кадрам (проверка CRC, игнорирование неверных пакетов без ответа).

### Ожидаемый результат

- Исходный код программы для МК на языке C.
- Схема подключения лампочек к портам МК (назначение выводов).
- Краткое описание протокола взаимодействия (адреса команд, коды ошибок).
